# Smart PDF Watermark Remover (智能PDF去水印工具)

这是一个基于 Python 开发的桌面应用程序，旨在通过智能分析 PDF 页面结构，识别并批量去除重复出现的“疑似水印”元素（包括图像和文本）。



## ✨ 功能特性

* **智能频率分析**：自动统计全文档元素，识别出现频率超过 **30%** 的疑似水印。
* **交互式确认**：在正式删除前，提供预览界面供用户勾选确认，支持按出现占比从高到低排序。
* **拖拽支持**：支持直接将 PDF 文件拖入程序窗口完成加载。
* **安全处理**：采用红线遮盖（Redaction）技术，确保文本清除彻底；图像删除则不影响文档其他部分。
* **自动命名**：清理后的文件默认以 `cleaned_原文件名.pdf` 命名另存。
* **多线程加速**：利用 Python 多进程并行分析文档结构，处理大文件更高效。
